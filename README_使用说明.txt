UK Order 价格查询工具 - 使用说明

一、日常怎么打开

1. 双击“启动工具.bat”。
2. 等待浏览器自动打开。
3. 如果浏览器没有自动打开，双击“打开页面.bat”。
4. 页面地址固定为：
   http://127.0.0.1:8501

如果工具已经在运行，再次双击“启动工具.bat”不会重复启动，会直接打开页面。

如果某台电脑打开中文 .bat 文件时出现乱码或命令错误，请改用英文入口：

start_tool.cmd      启动工具
stop_tool.cmd       关闭工具
restart_tool.cmd    重启工具
open_page.cmd       打开页面


二、日常怎么关闭

1. 使用完成后，双击“关闭工具.bat”。
2. 如果刚更新过工具文件，建议双击“重启工具.bat”。


三、导入价格资料

1. 打开页面后进入“导入工作台”。
2. 点击“选择文件夹”，选择存放 PDF 价格表的文件夹。
3. 可选：在“可选 Excel 文件”中选择一个或多个业务 Excel。
   - 不上传 Excel 也可以识别 PDF。
   - 上传 Excel 后，工具会把 PDF 和 Excel 一起汇总到查询表。
4. 可选：点击“扫描文件名”，先检查 PDF 文件是否能被识别。
5. 点击“开始识别并汇总”。
6. 等待进度条完成。


四、查询价格

1. 进入“汇总查询”。
2. 可以输入关键词，也可以按 Product Code、Tier、Size、Supplier、Currency、Cover Range 等筛选。
3. 表格最多先显示 1,000 行，导出时会按当前条件导出全部匹配结果。


五、复核异常

进入“异常复核”查看需要人工确认的数据，例如：

- PDF 价格为空。
- 价格超出常见范围。
- 尺寸无法自动识别。
- 同一产品存在多个版本。
- 同一查询键出现不同价格。

异常不代表一定错误，只表示建议人工看一眼来源文件。


六、导出 Excel

1. 进入“导出结果”。
2. 如果只想导出当前筛选结果，勾选“按当前查询条件导出”。
3. 点击“导出 Excel”。
4. 导出的文件会保存在 output 文件夹。


七、遇到问题怎么办

1. 页面打不开：
   - 先双击“打开页面.bat”。
   - 仍打不开时，双击“重启工具.bat”。

2. 启动失败：
   - 查看 logs/startup.log。
   - 常见原因是依赖没有安装完整，或端口 8501 被其他程序占用。

3. PDF 文件夹无法选择：
   - 使用“手动输入路径（备用）”。
   - 输入路径后点击“应用手动路径”。

4. 导入结果为空：
   - 先点“扫描文件名”，确认文件夹里确实有 PDF。
   - 确认 PDF 不是压缩包里的文件，需要先解压。


八、重要文件夹

app/       页面程序
src/       识别、清洗、导入、导出逻辑
config/    默认配置和映射规则
data/      本地数据库
output/    导出的 Excel
logs/      启动和运行日志
temp/      上传文件临时目录
